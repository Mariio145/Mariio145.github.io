About this template

This is Basic Noise, a free website template by Andreas Viklund. It offers a simple and useful layout that can be built upon and experimented with. The default style is 980 pixels wide, and pre-defined styles are available for fixed widths of 760 and 600 pixels, as well as a full-width (100%) layout (as in the Variant Multi template).

Free to download, free to use

Like all templates from andreasviklund.com, Variant Wide is free to use for any purpose.

For more information, tips and additional resources, visit andreasviklund.com.

Find the hidden features

The stylesheet includes a number of useful options for customization and additional layout variations. Alternate versions of this template will be made available to show examples of these options.
